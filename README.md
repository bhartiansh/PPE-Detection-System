# PPE Detection System

This project implements a real-time multi-camera PPE detection system using YOLO models (v5, v8, v11, v12).

## Features
- Multi-camera support (Laptop, Mobile, ESP32)
- Real-time detection
- Streamlit dashboard
- Model switching from UI
- Safety violation alerts

## Tech Stack
- Python
- OpenCV
- Streamlit
- Ultralytics YOLO

## How to Run

```bash
pip install -r requirements.txt
streamlit run dashboard.py
